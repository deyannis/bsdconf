Boot the installation media and drop to the shell by choosing "S"

Identify the disk that will be used to install the OS on

 $ dmesg | grep sd

In this example, we will use sd0

Device explanation: 
sdX:  the Xth SCSI-like disk (i.e.: SATA disks connected to AHCI interfaces, 
      USB drives, disk arrays connected to RAID controllers)
wd0:  the Xth IDE-like disk (i.e., IDE, SATA, etc., 
      connected to wdc or pciide interfaces)
rsdX: the Xth raw SCASI-like disk
rwdX: the Xth  raw SCASI-like disk

Partition explanation:
part a: boot disk root
b: boot disk swap
c: entire disk

Make the device for sd0

 $ cd /dev && sh MAKEDEV sd0

Purge the disk with random data

 $ dd if=/dev/urandom of=/dev/rsd0c 


Create GPT

$ fdisk -gy -b 960 sd0

$ disklabel -E sd0
	a a
	"enter"
	*
	RAID
	w
	q

$ bioctl -c C -l sd0a softraid0

add pass and remember the new device (sd2)

sh MAKEDEV sd2

overwrite the first MB
dd if=/dev/zero of=/dev/rsd2c bs=1m count=1

START INSTALL

exit
